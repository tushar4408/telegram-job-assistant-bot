# Telegram Job Assistant Bot

A Telegram bot that provides real-time job alerts based on user skills using Python, Telegram Bot API, and Adzuna Job API.

## Features
- Fetches live job listings automatically
- Skill-based job search
- Easy Telegram commands
- Uses Adzuna Job API

## Tech Stack
- Python
- Telegram Bot API
- Requests Library
- Adzuna API

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Create .env file and add:

TELEGRAM_TOKEN=your_token
ADZUNA_APP_ID=your_id
ADZUNA_APP_KEY=your_key

3. Run the bot:
python bot.py
